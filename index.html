👤 디자이너별 개인 채널

디자이너마다 개인 슬랙 채널 URL을 등록하면
본인 고객이 체크인할 때만 해당 채널로 알림이 갑니다.

📖 슬랙 Webhook 설정 방법

1
슬랙 앱 디렉토리에서 Incoming WebHooks 검색 후 설치
2
알림을 받을 채널 선택 (디자이너별로 DM 채널 따로 생성 권장)
3
생성된 Webhook URL 복사 (https://hooks.slack.com/services/...)
4
위 입력란에 붙여넣고 저장 → 테스트 발송으로 확인
⚠️ CORS 주의: 슬랙 Webhook은 브라우저에서 직접 호출 시 CORS 오류가 날 수 있어요. 이 경우 Chrome 확장 CORS Unblock 을 설치하거나, 실제 서비스 시엔 간단한 백엔드 프록시를 사용하세요.
